#EXPLANATION

### Kenapa menggunakan tampilan Ui seperti ini

```
  1. Untuk user yang sudah familiar terhadap komoditas ikan, jika data disajikan dalam bentuk tabel dan aksesnya menggunakan pc dilihatnya lebih mudah. Namun apabila user menggunakan hp, visualisasi tabel akan terpotong (butuh effort lebih untuk user melihat data - datanya karena harus di scroll secara intens)
```

```
  2. Biasanya orang lebih familiar dengan visualisasi berbentuk card dan tambahan illustrasi. Lebih mudah memahami apa maksud dari data yang disajikan. Ini bisa juga untuk menarik impression orang awam yang tidak sengaja visit ke web kita.
```

```
  3. Penambahan menu filter dan pencarian untuk memudahkan user dalam mencari data yang diinginkan.
```

```
  4. Menyesuaikan tampilan kekinian seperti layaknya list card pada e-commerce, dan bisa membuat orang cepat mengerti
```
